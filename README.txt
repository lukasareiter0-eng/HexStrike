BlueWhale GUI App

So funktioniert das Projekt:
1. Lege die gewünschte Waal-Logo-Datei als "logo.png" in denselben Ordner wie diese Dateien.
2. Starte die App mit Python:
   python main.py
3. Gib Text in das Eingabefeld ein und klicke auf "Speichern".
4. Der Text wird in "bluewhale_input.txt" im selben Ordner gespeichert.

Um eine EXE zu erzeugen, kannst du z.B. das Skript verwenden:
   build.bat

Wenn du das Icon erneut erzeugen möchtest, führe aus:
   python generate_icon.py

Hinweis: Das Logo wird oben links angezeigt und "BlueWhale" steht oben in der Mitte.
